### 1. CozyShop SQL Project
**Description:**  
A relational database project for a fictional business, CozyShop, which sells handmade booties. The project covers database schema design, table relationships, and SQL queries to manage customers, orders, and products.

**Highlights:**  
- Created tables for Customers, Orders, Products, and Inventory  
- Designed relationships using primary and foreign keys  
- Executed SQL queries for:  
  - Retrieving customer purchase history  
  - Calculating total sales  
  - Listing products low in stock

### 2. Teen Phone Use SQL Project
**Description:**  
A database project analyzing phone usage among teenagers. The project involves tables for Teens, Phone Usage, Apps, and Screen Time, allowing queries to study usage patterns, app popularity, and trends over time.

**Highlights:**  
- Created tables for Teens, Phone Usage, Apps, and Screen Time  
- Queries include:  
  - Average daily screen time by age group  
  - Most frequently used apps  
  - Identifying teens exceeding recommended phone usage limits
